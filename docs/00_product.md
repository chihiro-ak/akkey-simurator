# 00_product.md

## Product summary
アクリルキーホルダーの動きシミュレーションができるプロダクト。
ユーザーが画像をアップロードし、穴位置や金具を設定すると、完成イメージや揺れ方を確認できることを目指す。

## Target users
- 同人グッズを作成するイラストレーター
- アクリルグッズを制作するライバーやイベンター
- 入稿前に完成イメージを確認したい個人クリエイター

## Primary user problems
- アクリルキーホルダーのどこに穴をあけるとどのように動くのか、作成するまでわからない
- 完成後の見た目や揺れ方を事前に確認しづらい
- Blender などの 3D ツールを使わないと検証しづらく、制作ハードルが高い

## Business goals
- 2か月以内に100人に使ってもらう
- 初見ユーザーでも短時間でシミュレーションできるようにする
- 将来的に有料オプションや制作支援機能へ拡張できる土台を作る

## MVP scope
- 画像アップロード
- 穴位置調整
- 金具の選択
- 単体アクキーの簡易シミュレーション
- 完成イメージのプレビュー

## Core constraints
- Stack: Next.js / React / TypeScript / Tailwind CSS / shadcn/ui
- Deadline: 未定
- Team: 個人開発 + AIエージェント
- Non-goals:
  - 初期段階では本格的な物理シミュレーションは行わない
  - 初期段階では入稿データの完全自動生成は行わない
  - 初期段階では認証・課金機能は実装しない
