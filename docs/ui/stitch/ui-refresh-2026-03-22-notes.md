- Screen title: `Simulator Main UI (Desktop)`

- Screen title: `Simulator: Preview-First UI (Desktop)`

- Screen title: `Simulator: Comparison & Fine-Tuning (Desktop)`

- ねらい: 穴位置や金具違いを比較しながら決めやすくする
- 向いている場面: 将来的に比較検討や履歴の価値を伸ばしたいとき
