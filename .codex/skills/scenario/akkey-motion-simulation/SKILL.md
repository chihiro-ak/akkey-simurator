# akkey-motion-simulation

## 目的
このアプリ固有の
- 穴位置
- 重心
- 支点
- 金具や接続パーツ
- キーホルダーの揺れ方

を、実物の納得感に寄せて調整する。

## この skill を使う場面
- 穴位置を変えた時の静止姿勢が不自然
- ドラッグ方向や戻り方が不自然
- 金具が斜めになりすぎる、または接続が切れて見える
- 参照実装 `C:\illustlation\ホロクル11th\akkey-sim` に寄せたい

## このアプリで先に決めること
1. 支点 = プレビューアンカー直下の接続点
2. 穴位置 = 本体画像上の穴ハンドル位置
3. 重心 = 本体シルエットの不透明ピクセル分布を主に使う
4. 金具の接続点 = パーツ画像の最下端不透明ピクセルを基本にする
5. 金具の立ち方は、必要なら描画補正で扱う

## 切り分け順

### 1. 穴位置と静止姿勢
- 穴位置を中央、左右端寄りで見比べる
- まず「どの向きで止まるか」が納得できるかを見る
- 戻らない時は減衰より先に平衡角の向きを確認する

### 2. 重心と慣性
- 重心 1 点だけで足りないかを確認する
- 端穴ケースでは、質量分布や慣性の近似が必要かを検討する
- 金具の質量が本体を食いすぎていないかを確認する

### 3. ドラッグ
- 右へ引いたら右へ、左へ引いたら左へ傾くこと
- 穴ハンドルのドラッグと本体ドラッグは別経路で確認する
- 離した瞬間の角速度持ち越しが強すぎないか、弱すぎないかを見る

### 4. 減衰
- 中心へ戻った瞬間に止まりすぎないか
- 数往復させたい時は `damping` だけでなく `stiffness` と解放時の初速も見る
- 逆さま防止クランプが自然さを壊していないかを見る

### 5. 金具の見え方
- まず接続が切れて見えないことを優先する
- そのうえで、本体回転を部分的に打ち消す描画補正を検討する
- 金具の見た目補正は、物理モデルとは別責務として扱う

## 参照実装に寄せる時の注意
- 揺れ方の手触りは `C:\illustlation\ホロクル11th\akkey-sim` を参照してよい
- ただし金属パーツの伸び縮みはこのアプリでは入れない
- 参照実装の完全再現より、穴位置と重心に対して納得できる挙動を優先する

## 完了前の最低確認
- 穴位置中央での静止姿勢
- 穴位置端寄りでの静止姿勢
- 本体ドラッグの追従方向
- 解放直後から減衰完了まで
- 金具が接続されたまま自然に見えること

## 禁止事項
- 金具が離れて見える状態で close すること
- 穴位置と静止姿勢の対応が崩れたまま、揺れ感だけで pass 判定すること
- 重心の問題を係数だけで押し切ること
