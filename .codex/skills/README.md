# Active Skills

このリポジトリでは、以下の skill を本開発で使用する。

- `planning/backlog-refiner`: 要件や既存状況から backlog を整理する
- `planning/sprint-planning`: 次スプリントの対象、リスク、依存関係を決める
- `implementation/webapp-implement`: Webアプリ実装前の構造整理と実装補助
- `uiux/frontend-design`: UI 実装時のデザイン品質向上
- `uiux/audit`: UI 品質、アクセシビリティ、レスポンシブ、パフォーマンスの監査
- `quality/webapp-test-checklist`: Webアプリ向けのテスト観点整理
- `documentation/docs-sync`: 実装差分に応じた docs 更新整理
- `workflow/commit-message-writer`: コミットメッセージ作成補助

以下は本リポジトリでは使用しない。

- ティラノスクリプト専用 skill
- シナリオ、分岐設計専用 skill
- 微粒度すぎる UI 補助 skill
