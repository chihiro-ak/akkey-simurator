# feature-done rubric

Review each item as PASS / FAIL / PARTIAL.

## Requirement fit
- The implemented behavior satisfies all defined done criteria.
- Any scope cuts are explicitly documented.

## State coverage
- Default state exists.
- Loading state exists when asynchronous work is present.
- Error state exists and is understandable.
- Success state is visible when appropriate.
- Disabled state exists for unavailable actions.

## Regression awareness
- Affected shared files were reviewed.
- High-risk flows were sanity checked.
